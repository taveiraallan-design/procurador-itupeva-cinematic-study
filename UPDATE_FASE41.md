# Fase 41 — Visualizações Cinemáticas dentro da Apostila

Adiciona uma seção visual complementar ao final de cada tópico da aba Apostila.

## Inclui

- Visualizar o raciocínio
- Fluxo jurídico
- Árvore de decisão
- Mapa mental visual
- Comparação visual
- Cena aplicada à Procuradoria Municipal
- Causa e consequência
- Botões para copiar, transformar em revisão, enviar para flashcards e marcar como dominado

Implementação com React + CSS puro, sem bibliotecas pesadas, sem imagens externas e com layout responsivo.
