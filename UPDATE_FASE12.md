# Fase 12 — Expansão do Simulador Cinemático

Esta fase expande o Simulador Jurídico Cinemático para múltiplos fluxos além de Atos Administrativos.

## Novos fluxos adicionados

- Atos Administrativos
- Processo Administrativo Disciplinar
- Licitações
- Responsabilidade Civil do Estado
- Poderes Administrativos

## Principais melhorias

- Seletor de tema cinematográfico no topo do simulador
- Reset automático ao trocar de fluxo
- Mapa do processo dinâmico por tema
- Timeline e modo prova funcionando por fluxo selecionado
- Cenas reutilizadas e adaptadas para novos conteúdos jurídicos
- Decisões com caminhos corretos e pegadinhas específicas
- Conteúdo jurídico mais amplo para estudo de concurso

## Arquivos alterados

- src/data/cinematicActsFlow.js
- src/pages/CinematicSimulatorPage.jsx
- src/components/simulator/StepMap.jsx
- src/styles/globals.css

Build executado com sucesso usando `npm run build`.
