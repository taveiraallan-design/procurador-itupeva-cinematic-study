# Fase 8 — Flashcards Inteligentes

Implementado:

- Banco local de flashcards em `src/data/flashcards.js`
- Tela Flashcards funcional com decks, cards vencidos, todos, erros e dominados
- Revisão espaçada com Errei / Difícil / Bom / Fácil
- Cards automáticos gerados a partir dos erros das questões
- Cards pós-simulado gerados a partir dos temas fracos
- Histórico salvo no localStorage
- Dashboard atualizado com métrica de flashcards

## Teste rápido

1. Abra Flashcards.
2. Clique em Virar card.
3. Classifique com Errei, Difícil, Bom ou Fácil.
4. Erre algumas questões em Questões e volte para Flashcards > Erros.
5. Faça um simulado e volte para Flashcards para ver cards pós-simulado.
