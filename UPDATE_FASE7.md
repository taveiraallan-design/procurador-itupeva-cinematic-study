# Fase 7 — Revisão Inteligente

Esta fase transforma a aba Revisão em uma central dinâmica baseada em:

- erros do banco de questões;
- questões marcadas para revisão;
- tópicos fracos identificados em simulados;
- agendamento simples de revisão espaçada.

## Rodar

```bash
npm install --no-audit --no-fund
npm run dev -- --host 127.0.0.1 --port 5176
```

## Atualizar por cima da pasta atual

Na pasta `Downloads`, use:

```bash
cd ~/Downloads && unzip -o procurador-itupeva-cinematic-study-fase7.zip && cd procurador-itupeva-cinematic-study && npm install --no-audit --no-fund && npm run dev -- --host 127.0.0.1 --port 5176
```

Depois abra: http://127.0.0.1:5176
